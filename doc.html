Fluxx DAO AI Co-Pilot

An AI-powered assistant for DAO governance. Summarize proposals, draft documents, analyze voting trends, and get strategic insights.

Features

  • Interactive AI chat demo for DAO governance
  • Summarize complex DAO proposals
  • Draft governance documents
  • Analyze voting patterns and trends
  • Responsive design with TailwindCSS
  • Lottie animations for visual engagement

Setup Instructions

  1. Clone the project repository.
  2. Install dependencies: npm install
  3. Set your Gemini/OpenRouter API key in server.js.
  4. Run the backend server: node server.js
  5. Open index.html in your browser.

API Integration

The project uses a Node.js proxy to securely connect to the AI API:

POST /api/chat
Headers:
  Content-Type: application/json
  Authorization: Bearer YOUR_API_KEY
Body:
{
  "messages": [
    {"role": "system", "content": "You are AI Co-Pilot, a Web3/DAO governance assistant."},
    {"role": "user", "content": "User message here"}
  ]
}
Response:
{
  "text": "AI response text"
}

Frontend Usage

The AI Co-Pilot chat works on the frontend using the following JS structure:

  • Add user messages with addMessage("You", text)
  • Send messages to backend API via fetch("/api/chat")
  • Display AI responses in #chatMessages
  • Include typing indicators with Lottie animation
  • Enable demo suggestion buttons to prefill chat input
